💾 데이터 백업
강사·일정·공지·관리자 모든 데이터를 안전하게 보관하세요.
마지막 백업: 아직 백업하지 않음
📦 전체 백업 (JSON)
강사·일정·공지·관리자 모든 데이터(비밀번호 해시 포함). 나중에 완전 복원 가능합니다.
📊 엑셀 보고서
6개 시트로 정리된 보고서. 사람이 보기 편하고 인쇄·공유에 적합. (복원은 불가)
📥 백업에서 복원
이전에 받아둔 JSON 백업 파일에서 데이터를 복원합니다.
⚠️ 주의사항
• 복원 시 현재 Firebase의 모든 데이터가 백업 파일 내용으로 완전히 교체됩니다.
• 이 작업은 되돌릴 수 없습니다. 복원 전에 현재 데이터를 먼저 백업해두세요.
💡 백업 권장 사항
• 주 1회 정기 백업을 권장합니다.
• 백업 파일은 외부 저장소(Google Drive, USB, 외장하드 등)에 보관하세요.
• JSON 파일에는 비밀번호 해시값이 포함되니 비공개로 관리하세요.
• 데이터가 많은 변동(대량 등록, 삭제) 직후엔 별도 백업을 받아두세요.
• 7일 이상 백업하지 않으면 상단에 알림이 표시됩니다.
